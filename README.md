# 📚 Mihon → AniList Sync
> Automatically sync your reading progress from **Mihon** (formerly Tachiyomi) with **AniList** – supports `.tachibk` backup and `.csv` export.

Access the website here ➡️

 ## ✨ Features

- ✅ Support for **full backup** (`.tachibk`) – extracts the last read chapter.
- ✅ Support for **library export** (`.csv`) – greater compatibility with work titles.
- ✅ Updates or adds mangas to your AniList.
- ✅ Automatically sets status: `CURRENT` (if you read any chapter) or `PLANNING`.
- ✅ Clean, responsive interface with **real-time logs**.
- ✅ **Integrated translation** for 🇧🇷 Portuguese, 🇪🇸 Spanish, 🇯🇵 Japanese, and 🇺🇸 English.
- ✅ 100% **client-side** – no server, your data never leaves your browser.

## 📋 Prerequisites

- Account on [AniList](https://anilist.co/).
- AniList API access token (see how to obtain it below).
- A backup/export file generated by Mihon.

## 🚀 How to use

### 1. Obtain the AniList token

1. Access [AniList Developer Settings](https://anilist.co/settings/developer).
2. Click on **"Create New Client"**.
- **Name**: any name (e.g., "Mihon Sync").
- **Redirect URL**: `https://anilist.co/api/v2/oauth/pin`
- The other fields can be left blank.
3. After creating, copy the **Client ID**.
4. Build the authorization URL:
`https://anilist.co/api/v2/oauth/authorize?client_id=YOUR_CLIENT_ID&response_type=token`

Please provide the text you would like to have translated.
5. Paste this URL into the browser, authorize the application, and copy the **token** that appears on the page. Paste this URL in the browser, authorize the application, and copy the **token** that appears on the page.

> ⚠️ Keep the token safe – it allows you to modify your list on AniList.

### 2. Export your data in Mihon

- **Full backup (recommended)**:
More → Settings → Data and Storage → Create Backup
It generates a `.tachibk` file with all the details, including read chapters. However, it may happen that
add mangas that have not been added to the library, such as mangas available in the history

- **Simple export (CSV)**:
More → Settings → Data and Storage → Export Library
It generates a `.csv` with just the titles (without progress).

## ⚠️ Limitations and Warnings
API rate limit: AniList allows ~90 requests per minute. The script waits 1 second between each work, avoiding blocks.

Title matching: AniList searches for the original name (Japanese/English). If your work has a name in another language or from a fan scan, it may not be found.
Solution: Edit the title in Mihon to the official name before exporting.

Progress via CSV: .csv files do not contain the number of the chapter read. They will add the work only with the status PLANNING.
To take advantage of the progress, use .tachibk.
